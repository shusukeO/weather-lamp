# weather-lamp
天気によって色が変わるランプ。  (2019年5月作成)

例えば、晴れのち雨だったら、晴れの赤と雨の青が混ざった色に光る。  

玄関に置いておくと、今日は傘が必要かどうかなどが直感的に分かる。
